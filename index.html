GenerationStandard — Guide complet

Application de génération et gestion des standards de travail

C'est quoi ?

GenerationStandard est une application mobile et PC qui permet de créer, corriger, valider et gérer les fiches d'instructions de travail (format PHAREC) de votre atelier.

Elle utilise l'intelligence artificielle pour générer un premier brouillon à partir d'une description simple, que vous corrigez ensuite avec le méthodiste.

Accès : ouvrez l'URL de votre atelier dans Chrome sur mobile ou PC. L'app fonctionne comme une vraie application — installez-la sur votre écran d'accueil.

Installation sur téléphone (Android)

1. Ouvrez l'URL dans Chrome

2. Menu 3 points en haut à droite → "Ajouter à l'écran d'accueil"

3. L'icône apparaît sur votre bureau

4. L'app s'ouvre en plein écran, comme une vraie application

Sur iPhone : ouvrez dans Safari → bouton Partager → "Sur l'écran d'accueil"

Les 4 écrans principaux

📚 BIBLIO — La bibliothèque

Liste de tous vos documents. Chaque document affiche :

  • Sa référence (ex: STD-71902605-001)
  • Son titre
  • Son statut (voir ci-dessous)
  • Ses tags

Filtres disponibles : recherche par texte, filtre par statut.

✨ NOUVEAU — Créer un document

1. Choisissez le type de document (Fiche poste / Standard qualité / Procédure)

2. Saisissez le poste (ex: 7190), l'opérateur, la description de l'opération

3. Cliquez Générer → l'IA rédige le brouillon complet en 15-30 secondes

4. Le document s'ouvre automatiquement pour correction

📄 DOCUMENT — Consulter et corriger

L'écran principal du document avec ses onglets :

  • 📄 Document — la fiche PHAREC avec toutes les étapes
  • ℹ️ Infos — référence, date, opérateur, lien qualité
  • 🤖 IA — demander des modifications à l'IA
  • 🎓 Formations — registre des formations ISO 9001
  • 📤 Export — télécharger en PDF ou Word
  • 🕐 Historique — toutes les modifications tracées

⚙️ PARAMS — Paramètres

Configuration de l'app, clé API, proxy, QR code, export/import de la bibliothèque.

Les statuts d'un document

Les documents suivent un cycle de vie précis :

📝 BROUILLON

Document généré par l'IA, pas encore corrigé.

Bouton disponible : ✏️ → Méthodes (une fois votre correction terminée)

🟠 CORRECTION TERRAIN

Vous êtes en train de corriger le document.

Ce que vous faites : corriger les descriptions d'étapes, ajouter les photos, ajuster les points clés S/Q/T.

🟡 EN ATTENTE MÉTHODES

Votre correction est terminée. Le document a été envoyé au méthodiste.

Boutons disponibles :

  • 📤 Doc méthodes → télécharge le fichier Word pour le méthodiste
  • 📥 Import méthodes → importer le fichier corrigé par le méthodiste
  • ↩ Renvoyer → si besoin de retravail

🟣 CORRECTION MÉTHODES

La correction du méthodiste a été importée.

Bouton disponible : ✅ Valider → finalise le document

✅ VALIDÉ

Document approuvé. Peut être utilisé en production.

Bouton disponible : ⭐ Référence → marquer comme exemple pour l'IA

⭐ RÉFÉRENCE

Document exemplaire. L'IA s'en inspire pour les prochaines générations.

Maximum 5 documents de référence par type.

Créer un document étape par étape

1. Générer le brouillon

Allez sur NOUVEAU :

  • Type : Fiche poste (pour les opérations courantes)
  • Poste : numéro du poste (ex: 7190)
  • Opérateur : nom (ex: MARTIN T.)
  • Description : décrivez l'opération en langage naturel
Exemple : *"Préparation du duo avant levage du caisson. Tarauder les vis de verrouillage de la chappe d'accroche. Nettoyer les trous taraudés. Appliquer la graisse sur l'emplacement porte. Installer les bras des vérins."*

Cliquez Générer → l'IA rédige toutes les étapes avec points clés S/Q/T.

2. Corriger le document

L'IA génère un premier jet — il faut le vérifier et corriger :

Description des étapes : cliquez sur le texte, modifiez directement.

Point clé S/Q/T : cliquez sur le bouton coloré (+ S / ◆ Q / ■ T) → un menu apparaît avec les 3 options :

  • + S — Sécurité : point critique pour la sécurité de l'opérateur
  • ◆ Q — Qualité : critère de contrôle qualité
  • ■ T — Technique : précision technique importante

Photos : cliquez sur "📷 Ajouter" dans une case → prenez plusieurs photos si besoin. Cliquez sur une miniature pour l'afficher en grand. Croix rouge pour supprimer.

Note libre : zone de texte sous les photos — écrivez ce que vous voulez (remarque, condition particulière...).

Ajouter une étape : bouton "+ Étape" en bas du tableau.

Supprimer une étape : croix ✕ à gauche du numéro.

3. Envoyer au méthodiste

Quand votre correction est terminée :

1. Cliquez ✏️ → Méthodes

2. L'app télécharge automatiquement un fichier Word (.doc)

3. Envoyez ce fichier par mail au méthodiste

4. Le statut passe à EN ATTENTE MÉTHODES

4. Importer la correction du méthodiste

Quand le méthodiste vous renvoie son fichier corrigé :

1. Ouvrez le document dans l'app

2. Cliquez 📥 Import méthodes

3. Sélectionnez le fichier .doc, .docx ou .txt renvoyé

4. L'app compare et applique les corrections automatiquement

5. Le statut passe à CORRECTION MÉTHODES

5. Valider définitivement

Vérifiez les corrections importées du méthodiste, puis :

1. Cliquez ✅ Valider

2. Le document est VALIDÉ — il peut être utilisé en production

3. La version est incrémentée automatiquement (v2, v3...)

6. Marquer comme référence (optionnel mais recommandé)

Si ce document est un bon exemple de ce qu'on attend :

1. Cliquez ⭐ Référence

2. L'IA l'utilisera pour améliorer les prochaines générations

3. Après 10 documents de référence, la qualité des brouillons augmente significativement

Exporter un document

Dans l'onglet 📤 Export :

📄 Télécharger FI-00 (impression)

Génère le document au format PHAREC exact, prêt pour impression A4.

Ouvrez le fichier téléchargé dans Chrome → Ctrl+P → "Enregistrer en PDF".

📝 Télécharger .doc (réécrivable)

Fichier Word que vous pouvez modifier dans Word ou LibreOffice.

C'est ce fichier que vous envoyez au méthodiste.

📋 Registre formation PDF

Liste de toutes les formations enregistrées sur ce document (ISO 9001 §7.2).

✉️ Envoyer par mail

Ouvre votre client mail avec le résumé du document pré-rempli.

Enregistrer des formations

Onglet 🎓 Formations :

1. Saisissez le nom de l'opérateur formé

2. La date (aujourd'hui par défaut)

3. Le nom du formateur

4. Cochez "Signé" si la fiche papier est signée

5. Cliquez + Enregistrer

Les formations sont exportables en PDF pour les audits ISO 9001.

Demander une modification à l'IA

Onglet 🤖 IA :

Décrivez ce que vous voulez modifier en langage naturel.

Exemples :
- *"Ajoute une étape de contrôle visuel avant le serrage"*
- *"Reformule l'étape 3 en insistant sur la sécurité"*
- *"Les temps alloués sont trop courts, augmente-les de 20%"*

L'IA met à jour le document directement.

Sauvegarder et restaurer la bibliothèque

Dans PARAMS → Bibliothèque :

⬇️ Exporter la bibliothèque

Télécharge un fichier JSON avec tous vos documents et photos.

À faire régulièrement comme sauvegarde.

⬆️ Importer une sauvegarde

Restaure une bibliothèque depuis un fichier JSON.

Les documents existants sont mis à jour, les nouveaux sont ajoutés.

Permet de transférer la bibliothèque vers un autre appareil.

Partager l'accès avec un autre appareil

QR code de l'app

Dans PARAMS → QR Code :

1. Saisissez l'URL de l'app

2. Cliquez QR → un QR code s'affiche

3. Scannez depuis l'autre appareil → l'app s'ouvre

4. Téléchargez le QR en PNG pour l'imprimer et le coller au poste

QR code de la clé API

Si un opérateur n'a pas la clé API sur son téléphone :

1. Depuis un appareil qui l'a → PARAMS → 📱 QR de ma clé

2. Depuis l'autre appareil → PARAMS → 📷 Scanner une clé

3. Pointez la caméra → clé importée automatiquement

Thèmes visuels

La clé API est automatiquement récupérée depuis le serveur Cloudflare à chaque ouverture de l'app. Vous n'avez pas à la ressaisir.

Questions fréquentes

"Erreur de génération — impossible de contacter l'API"

→ Vérifiez votre connexion internet.

→ Dans PARAMS : vérifiez que l'URL du Worker Cloudflare est bien saisie.

→ La clé API sera récupérée automatiquement depuis le serveur.

"Le document généré ne ressemble pas à ce qu'on veut"

→ Corrigez-le, validez-le, marquez-le ⭐ Référence.

→ Les prochaines générations s'amélioreront automatiquement.

"J'ai perdu mes documents"

→ Les données sont dans le navigateur. Si vous avez exporté une sauvegarde JSON → importez-la dans PARAMS.

→ Si pas de sauvegarde → les données ne sont pas récupérables. Faites des exports réguliers.

"Comment retrouver un ancien document ?"

→ Bibliothèque → barre de recherche → tapez le poste, l'opérateur ou des mots clés.

"Le méthodiste ne peut pas ouvrir le fichier"

→ Essayez de lui envoyer le format .doc (plus universel que .docx).

→ Il peut aussi l'ouvrir dans Google Docs ou LibreOffice (gratuit).

Conseils pratiques

  • Décrivez précisément : plus la description initiale est détaillée, meilleur est le brouillon
  • Ajoutez des photos dès la correction : une photo vaut mieux qu'une description longue
  • Marquez les références régulièrement : l'IA s'améliore à chaque document validé
  • Exportez la bibliothèque une fois par semaine comme sauvegarde
  • Utilisez les tags pour retrouver facilement les documents par opération ou type
